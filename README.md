# Kali NetHunter Installation in Termux
- Repositori ini memungkinkan Anda untuk menginstal Kali NetHunter di Termux menggunakan skrip otomatis. Dirancang untuk pemula yang menginginkan lingkungan Kali Linux portabel di Android.

## Persyaratan

- Perangkat Android (Tidak memerlukan root)
Aplikasi Termux (versi terbaru dari F-Droid)
Koneksi internet yang stabil
Ruang penyimpanan internal minimal 5GB.

## Fitur

- Skrip instalasi sekali klik
- Menginstal semua dependensi yang dibutuhkan
- Penanganan kesalahan otomatis disertakan

### Termux Full Install and Setup 
```
pkg update && pkg upgrade && pkg install git && git clone https://github.com/shopeebjm/install-kali-termux.git && cd install-kali-termux && chmod +x install-kali.sh && ./install-kali.sh
```

### [TERMUX] Packege Install Comments

```
pkg update && pkg upgrade -y
```
```
pkg install git -y
```
```
git clone https://github.com/shopeebjm/install-kali-termux
```
```
cd install-kali-termux
```
```
chmod +x install-kali.sh
```
```
./install-kali.sh
```

### Setelah instalasi selesai, jalankan perintah ini.
```
nethunter
```
```

```
